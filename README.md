# OpenWrt-Auto
基于官方openwrt 编译 x86 固件


# 说明  
本固件高度自定义，贴近个人使用场景。      
 - 基础上网功能（默认 OpenWrt 配置）
 - 默认支持 ipv6      
 - 添加 UPNP
 - 添加主题 Argon      
 - 添加 磁盘管理 Diskman      
 - 添加 挂载功能
 - 添加三方 Docker
 - eth0 默认 WAN 口
